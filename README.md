# Lead-Gen-Google-Sheet-API-Galvan-AI

A secure, robust, and highly flexible Lead Generation CRM engine built directly inside Jupyter Notebook. This project connects programmatically to the Google Sheets V4 API using OAuth 2.0 user credentials, featuring a localized data validation pipeline (via Pandas) and comprehensive error handling modules.

## 🚀 Key Features

* **OAuth 2.0 Personal Authentication:** Acts directly as your user profile without needing to share files with external bot emails. Uses token caching (`token.pickle`) to skip the login screen after the first authorization.
* **Dual-Layer Verification Engine:** Validates data *locally* in Python before it hits the network, preserving your Google API call limits.
* **Strict Structural Data Validation:**
    * **Phone Numbers:** Enforces a precise regex pattern (`XXX-XXXX` - exactly 7 digits with a single dash).
    * **Emails:** Structural syntax verification requiring an `@` sign and a valid domain extension.
    * **Categories:** Restricts inputs to approved marketing channels and pipeline statuses.
* **Partial Batch Commits:** Automatically separates clean records from corrupted rows during batch operations, pushing valid data to the cloud while flagging bad entries in a local quarantine viewer.
* **Advanced Dynamic Lookup Engine:** Search, filter, and isolate records instantly by **Row Number**, **Column Name**, **Column Letter (A-Z)**, or **1-Based Column Number Index**.
